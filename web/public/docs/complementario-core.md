## Trabajo Complementario y Core

### Unilateral (elegir 2 por sesión, rotar):
- Zancadas caminando con mancuernas: 3x10-12 cada pierna
- Curl bíceps con mancuerna: 3x12-15 cada brazo
- Extensión tríceps sobre cabeza con mancuerna: 3x12-15
- Elevaciones laterales con mancuerna: 3x15-20
- Press banca inclinado con mancuerna: 3x10-12

### Core (elegir 2 por sesión):
- Plank frontal: 3x30-45s
- Plank lateral: 3x20-30s cada lado
- Dead bug: 3x12-15 cada lado
- Russian twist con mancuerna ligera: 3x20-30 (10 cada lado)
- Elevación de piernas colgado: 3x10-15
